# Gillsystems.net 2025 Reboot

A modern, living portfolio for Gillsystems, showcasing 30+ years of technology expertise.

## Mission
"Systems Should Serve Humans." We reject vendor lock-in and black boxes. We build transparent, empowering systems.

## Features
- **Legacy Toggle**: Switch between the modern 2025 terminal aesthetic and the 2010 legacy style.
- **Live GitHub Activity**: Real-time feed of open source contributions.
- **Evolution Timeline**: Interactive journey through 30+ years of tech evolution.
- **Services**: Value Engineering and Legacy Modernization.
- **Zero Fluff**: Built with Vite + Vanilla JS for maximum performance and longevity.

## Tech Stack
- **Framework**: Vite (Vanilla JS)
- **Styling**: CSS Variables (No frameworks, no bloat)
- **Deployment**: GitHub Pages (Automated via Actions)

## Local Development
1. Install Node.js.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open the local URL.

## Deployment
This repository includes a GitHub Action to automatically deploy to GitHub Pages.
1. Go to Settings > Pages.
2. Under "Build and deployment", select **GitHub Actions** as the source.
3. Push to `main`.
